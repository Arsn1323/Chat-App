# 💬 Chat App

A full-stack real-time chat application built with:

- ⚛️ **Frontend**: React + Vite + Tailwind CSS
- 🧠 **Backend**: Node.js + Express + MongoDB
- 🚀 **Deployment**: 
  - Frontend on [Vercel](https://vercel.com)
  - Backend on [Render](https://render.com)
